# Hướng dẫn Thực hành - Điện tử Số

Xem file concepts.md để hiểu lý thuyết trước khi thực hành.

## Dự án thực hành

### Dự án 1: Bộ đếm 7-Segment
- IC 7490 + 7447 + 7-segment
- Đếm từ 0-9
- Hiển thị số

### Dự án 2: Shift Register 74HC595
- Điều khiển 8 LED với 3 chân Arduino
- Hiệu ứng LED chạy
- Cascading nhiều IC

### Dự án 3: LED Matrix 8x8
- Module MAX7219
- Hiển thị ký tự và hình ảnh
- Animation đơn giản

### Dự án 4: Đèn giao thông
- State machine logic
- Timer tự động
- Mô phỏng đèn 2 hướng
