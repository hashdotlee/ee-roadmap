# Hướng dẫn Thực hành - Chuyên sâu

Xem file concepts.md để hiểu lý thuyết trước khi thực hành.

## Dự án thực hành

### Dự án 1: IoT với ESP32
- Đọc cảm biến DHT22
- Gửi dữ liệu lên ThingSpeak
- Hiển thị trên web/app

### Dự án 2: NRF24L01+ Wireless
- Truyền dữ liệu giữa 2 Arduino
- Điều khiển từ xa
- Tầm xa 100m

### Dự án 3: ESP32 Web Server
- Tạo server trên ESP32
- Điều khiển LED/relay từ trình duyệt
- Giao diện web đơn giản

### Dự án 4: Robot tránh vật cản
- Cảm biến Ultrasonic HC-SR04
- Driver động cơ L298N
- Logic tự động

Xem file concepts.md để biết chi tiết về các giao tiếp và wireless.
